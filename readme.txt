A seamless and secure payment gateway integration for WooCommerce using TZSMM Pay.

== Description ==

The **TZSMM Pay Gateway Plugin** allows you to integrate TZSMM Pay as a payment method on your WooCommerce store. It ensures secure transactions, easy setup, and a user-friendly experience for both the store owners and customers. 

**Features:**
* Secure payment processing.
* Easy integration with WooCommerce.
* Support for multiple currencies.
* Real-time payment status updates.
* Intuitive settings and API configuration.

[Live Demo](https://tzsmmpay.com)

[Need Help?](https://t.me/tzsmmpay)

== Features ==

- Accept secure payments via TZSMM Pay.
- WooCommerce-compatible with real-time payment updates.
- Supports multiple currencies and seamless integration.
- Fully automated transaction handling.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/tzsmmpay-gateway` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Settings > Payments, and enable "TZSMM Pay Gateway".
4. Configure the API Key and other settings as needed.

== Frequently Asked Questions ==

= What is the API Key? =
The API Key is provided by TZSMM Pay and is required to authenticate your WooCommerce store with the TZSMM Pay payment gateway.

= How do I contact support? =
For support, visit our [Telegram channel](https://t.me/tzsmmpay) or [website](https://tzsmmpay.com).

= Does this plugin support refunds? =
Currently, refunds need to be processed manually via the TZSMM Pay dashboard.
