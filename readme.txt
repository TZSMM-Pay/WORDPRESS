== Installation ==  

1. Upload the plugin files to the `/wp-content/plugins/tzsmmpay-gateway` directory, or install it through the WordPress plugins screen directly.  
2. Activate the plugin through the 'Plugins' screen in WordPress.  
3. Navigate to WooCommerce > Settings > Payments and enable "TZSMM Pay Gateway".  
4. Configure the API Key and other settings as per your account on TZSMM Pay.  

**Troubleshooting:**  
- Ensure your PHP version is 7.4 or higher.  
- Verify that WooCommerce is installed and activated before activating this plugin.  

== Frequently Asked Questions ==  

= What is the API Key? =  
The API Key is provided by TZSMM Pay and is required to authenticate your WooCommerce store with the TZSMM Pay payment gateway.  

= How do I contact support? =  
For support, visit our [Telegram channel](https://t.me/tzsmmpay) or [website](https://tzsmmpay.com).  

= Does this plugin support refunds? =  
Refunds need to be processed manually via the TZSMM Pay dashboard.  

= Does this plugin support recurring payments? =  
Currently, recurring payments are not supported. Stay tuned for future updates.
